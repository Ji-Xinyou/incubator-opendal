Detailed Description

The opendal_operator_ptr is used to access almost all OpenDAL APIs. It represents a operator that provides the unified interfaces provided by OpenDAL.

See also

Note
The opendal_operator_ptr actually owns a pointer to a opendal::BlockingOperator, which is inside the Rust core code.
